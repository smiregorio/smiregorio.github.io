Column specifications:
  • pid - assigned party id,
  • cntr - country name,
  • pname1 - party name in English,
  • pname2 - party name in the original language
  • votes - last national election result
  • mandates1 - current number of mandates in the parliament (including changes after an election)
  • total1 - maximum seat capacity of the parliament
  • date - last national election date, DMY format
  • next - next election year
  • gov - current government status (including changes after an election)
  • mandates2 - current number of mandates in the EU parliament (including changes after an election)
  • total2 - number of seats assigned to the country at the EU parliament
  • eugroup - current EU parliamentary group or party if applicable
Notes:
